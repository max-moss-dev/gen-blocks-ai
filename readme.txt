Create editable WordPress blocks with AI

=== Description ===

Gen Blocks is a plugin that allows you to create editable WordPress blocks with AI. 
It uses Anthropic's Claude 3.5 Sonnet API to generate the blocks. 
The plugin provides a simple interface for creating blocks, 
and simplified all-in-one-block editing experience.

=== Installation ===

1. Upload the plugin to your WordPress site.
2. Activate the plugin.
3. Activate the license key in the "Gen Blocks" menu.
4. Go to settings page under "Gen Blocks" and enter your Anthropic API key.
3. Go to the "Add New Block" menu to create your first block.

=== Usage ===

1. Go to the "Add New Block" menu to create a new block.
2. Describe the block you want to create to the "Instructions" field.
3. Attach an image of block you want to create (Optional).
4. Click the "Generate Block" button to generate the block.
5. After the block is generated, you can adjust the block via the chat UI.
6. When the block is done, click the "Save Block" button to save the block.
7. You can then use the block in your page or post via the blocks UI.
